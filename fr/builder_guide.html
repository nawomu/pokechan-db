Comment utiliser le Team Builder

Choisissez vos Pokémon, capacités et statistiques pour composer votre équipe

Le Team Builder est un outil tout-en-un pour composer une équipe solide. Choisissez un Pokémon, explorez les capacités qu'il peut apprendre et définissez les statistiques de base et les statistiques réelles (nature / EV) pour vérifier la puissance de votre Pokémon et la répartition de vos points. Garder cette page ouverte pendant un combat vous aide aussi à anticiper les capacités que le Pokémon adverse pourrait posséder.

1Choisir un Pokémon pour un emplacement

Choisissez un Pokémon depuis « ＋ Sélectionner » dans chaque emplacement. Vous pouvez rechercher par nom ou par numéro de Pokédex, et les Méga-Évolutions ainsi que les formes régionales sont également disponibles.

2Choisir les capacités

Cliquez sur la zone des capacités sous le nom d'un Pokémon pour ouvrir la liste des capacités (version Team Builder). Cochez les capacités souhaitées parmi celles qu'il peut apprendre, puis validez.

3Définir les statistiques

Cliquez sur la zone des statistiques pour ouvrir la fenêtre de réglages. Choisissez une nature et répartissez les EV entre chaque statistique : les statistiques réelles sont calculées instantanément. Les boutons MIN / MAX permettent de pousser une valeur au maximum rapidement.

4Basculer l'affichage et déplacer les emplacements

Utilisez les boutons « Base / 6V / Nature / EV252 / Max / Rang » pour afficher ou masquer chaque ensemble de valeurs. Vous pouvez aussi faire glisser un emplacement pour le déplacer vers un autre emplacement.

# Kimi K3 セットアップメモ (2026-07-17調査)

Kimi K3(Moonshot)を GLM と同じ「実装役」として Claude Code CLI に乗せるための手順と料金。

## 料金の全体像(2026-07-17時点)

**★大前提: 「月額プラン」と「API」は完全に別システム(合算されない)**

### ① API従量課金(Claude Code乗せ換えで使うのはこっち)
| 項目 | 料金(per 1M tokens) |
|---|---|
| 入力(キャッシュミス) | $3.00 |
| 入力(キャッシュヒット) | $0.30 |
| 出力 | $15.00 |

- 1Mコンテキスト全域で均一料金・Web検索は1回$0.004
- 最低チャージ $1 から。累計$5チャージで$5バウチャー
- **7/15〜8/11: チャージ10〜30%ボーナスキャンペーン中**
- 参考: Claude Sonnet 5 = $3/$15(〜8/31は導入価格$2/$10)と同帯。GLM Coding Lite($16.2/月定額)とは課金方式が違う

### ② 月額プラン(チャット/Kimi Code用・APIとは別)
| プラン | 月額 | 内容 |
|---|---|---|
| Adagio | 無料 | 基本チャット・軽量モデル |
| Moderato | $19 | Deep Research・Kimi Code入門・Slides等 |
| Allegretto | $39 | Moderatoの上位クォータ |
| Allegro | $99 | Agent Swarm |
| Vivace | $199 | 300-agent swarm・最大クォータ |

- 「Kimi Code」= Moonshot自身のコーディングCLI(Claude Code相当品)。OAuthログインで月額プランから使える
- **Claude Code CLIへの乗せ換えは公式ドキュメントではAPIキー(=①従量課金)前提**。月額プランでClaude Codeが賄えるという記載は無し(GLMのCoding Liteとはそこが違う)

## セットアップ手順(阿部さんの作業)

1. https://platform.kimi.ai (Kimi Open Platform)でアカウント作成
2. 少額チャージ($1〜。キャンペーン中なのでボーナス付き)
3. デフォルトプロジェクトでAPIキーを発行
4. キーを保存: `echo 'sk-xxxx' > ~/.kimi_key && chmod 600 ~/.kimi_key`
5. `~/.zshrc` に下記を追記(glm関数の下):

```zsh
# Kimi K3版 Claude Code (実装役候補・2026-07-17) — kimi で起動
kimi() {
  ANTHROPIC_BASE_URL="https://api.moonshot.ai/anthropic" \
  ANTHROPIC_AUTH_TOKEN="$(cat ~/.kimi_key 2>/dev/null)" \
  ANTHROPIC_MODEL="kimi-k3" \
  ANTHROPIC_DEFAULT_OPUS_MODEL="kimi-k3" \
  ANTHROPIC_DEFAULT_SONNET_MODEL="kimi-k3" \
  ANTHROPIC_DEFAULT_HAIKU_MODEL="kimi-k3" \
  CLAUDE_CODE_SUBAGENT_MODEL="kimi-k3" \
  ENABLE_TOOL_SEARCH="false" \
  claude "$@"
}
```

(環境変数は公式ドキュメント https://platform.kimi.ai/docs/guide/claude-code-kimi のとおり。`ENABLE_TOOL_SEARCH=false` も公式指定)

6. 新しいターミナルで `cd "/Users/masamichi/Documents/ポケモンDB" && kimi`

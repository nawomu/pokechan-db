## 1. 今日やったこと(時系列・確定済み=動作確認OK)

すべて `real_battle_simulator.html`。各変更後に「インラインscriptのSyntaxErrorチェック＋`node tools/_sim_test.js`(63/63)」を毎回実施。

1. **技スロットの✕削除ボタン追加 / 技名クリックでモーダルを開かない**(のち #9 で統合され不要に)。
2. **技プールに並び替えチップ**(威力順/タイプ)＋**カテゴリ絞り込み**(物理/特殊/変化トグル)。
3. **プール各行に情報表示**(威力・命中・PP・説明)。最初は2〜3段→**1行**に圧縮(横並び、説明は省略+ホバー)。
4. **ポケモン選択モーダルをクリックしたカードの近く(中央)にアンカー表示**(`positionModalBox`)。マウス移動削減。
5. **起動時デフォルト編成**(フシギバナ vs リザードン)を自動セット(`DEFAULT_SETUP` / `applyDefaultSetup`)。リロードのたびに選び直す手間をなくす。
6. **プール全技に対相手ダメージ(HP%・確定/乱数発数)を最初から表示**。
7. **ダメージ表示を縦2段グリッド→横1行に圧縮**(`dmg-line`)。
8. **★大改修: 「上のスロット4枠＋下のプール」を廃止し、全技を1リストに統合。**
   - `st.moves = usableMoves(poke)`(覚える全技、正規順)。`selectedMoveIdx` はその**正規index**。
   - 各行に **選択/実行** ボタン＋情報＋ダメージ。**選択中の行の下だけ詳細ダメージ行**を展開。
   - 並び替え/絞り込み/検索はリスト上部。`moves-scroll`(max-height 460px)でスクロール。
   - 選択しても**スクロール位置を保持**。検索キーワードは `st.moveSearch` で再render後も維持。
   - `st.moves` を全技にする箇所: `makeSideState` / モーダル選択 / `applyDefaultSetup` / `applyHandoffToSide`。
9. **バトルログを中央カラムへ移動**＋**中央を 210px→340px に拡張**(`#arena` grid)。旧 `#battle-panel`(下の全幅ログ)は撤去。
10. **技説明のホバーポップアップ**(`#move-tip` / `setupMoveTooltip`)。`data-tip` を**行全体**に付与=行のどこに乗せても全文表示。native title は廃止。
11. **行のどこをクリックしても選択中に**(ボタン以外)。hoverハイライト＋pointerカーソル。
12. **ボタンの色/ラベル調整**: 「実行」から ⚔ を削除(×印に見えるとの指摘)/「選択」を**80年代グリーンスクリーン風の緑**/「実行」を**ベージュ系**(激しいオレンジをやめる)。
13. **ダメージ表示の重複解消＋日本語化**:
    - 選択中の行はミニ表示(HP%/発数)を出さず、下の詳細1行に集約(2重表示をやめる)。
    - チップ **`STAB`→「タイプ一致」** / **「1/4 軽減 ×0.25」→「こうかいまひとつ ×1/4」**(×0.5→×1/2 等、分数表記)。
    - 「ダメ 19〜23」+別枠HP% → **「ダメージ 19〜23（HP 12.4〜15.0%）」** に統合。
14. **「ダメージ順」並び替えを追加し、既定に**。相手に実際に与える**最大ダメージ順**(タイプ一致1.5倍＋相手の相性込み)。当たらない/変化技は下。`dmgCache`/`getDmg` で1回だけ計算し並びと表示で共用。相手未選択時は威力順にフォールバック。

## 2. いまのコード構造メモ(次の人が触る前に)

- **side state(`makeSideState`)** に追加済み: `poolSort`('damage'|'power'|'type', 既定 damage) / `poolFilter`(null|物理|特殊|変化) / `moveSearch`(検索語)。`moves` は**覚える全技**, `selectedMoveIdx` はその正規index。
- **技リスト描画 = `renderMoves(root, st)`**。`dmgCache`/`getDmg(m)` で対相手ダメージをキャッシュ。`rowHtml` が1行。`dmgMiniHtml`(行の at-a-glance) / `dmgFullHtml`(選択中の詳細1行) / `dmgKoLabel` / `dmgKoClass`。並びは mode==='damage' のとき `dmgSortVal` 降順、その他は `sortMoveItems`。
- **検索** = `applyMoveRowSearch(wrap, q)`(行を表示/非表示)。input で `st.moveSearch` に保存→再render後 再適用。
- **ツールチップ** = `#move-tip`(CSS) / `setupMoveTooltip()`(document に mouseover/mouseout/scroll 委譲、`[data-tip]` を拾う)。boot で呼ぶ。
- **デフォルト編成** = `const DEFAULT_SETUP`(self/opp の name と moves) / `applyDefaultSetup()`(boot, render前に呼ぶ)。希望技名→なければ威力上位で補完(現仕様は全技表示なので先頭technだけ選択)。
- **デッドコード**: `openMovePickerModal`(waza-list.html iframeの技ピッカー)はもう誰も呼ばない(全技リスト化で不要)。残置=害なし。将来掃除可。
- **レイアウト**: `#arena` = `grid 1fr 340px 1fr`。中央 `#env-panel` に 環境/攻撃側設定/防御側設定/バトル操作/📜バトルログ。

## 📋 収集すべき情報

### A. ダメージ計算系持ち物 (優先度: 高)

各持ち物について以下を埋める:

```json
{
  "key": "kodawari_hachimaki",
  "name": "こだわりハチマキ",
  "category": "attack_boost",
  "effect": "物理技の威力 ×1.5",
  "q12": 6144,
  "factor": 1.5,
  "applies_to": "physical",
  "restriction": "技を1つに固定",
  "source": "vgc-champions-calc",
  "notes": ""
}
```

**収集対象**:
1. **威力補正系**: こだわりハチマキ / こだわりメガネ / いのちのたま / たつじんのおび / ちからのハチマキ / ものしりメガネ / プレート各種 (ほのおのジュエル等は廃止？要確認)

4. **特殊効果**: とつげきチョッキ / しんかのきせき / ふうせん / きあいのタスキ / ヨプのみなど

### vgc-champions-calc バンドル再解析の手順

```bash
# ブラウザの DevTools → Network → ハッシュ付き JS ファイル (`/assets/index-*.js`) を保存
# その中で以下のキーワードを grep
grep -o "アイテム名\|kodawari\|hachimaki\|chiyokoreto" assets/index-*.js
# Q12 値は 6144 / 4915 / 5324 / 2048 などで定数として出現
```

## ⚠️ 注意

- **ポケチャンは Sword/Shield や SV と仕様が違う** — 「テラスタル無し / Zクリスタル無し / ダイマックス無し」を前提に、登場しないアイテムは除外する。**メガシンカは実装済み (60種)** なので注意。
- **Champions 新規アイテム** がある場合は「source: champions_only」とマークする

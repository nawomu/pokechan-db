## 🚦 注意点

### 1. 共有領域への touch

`index.html` / `index_en.html` / `i18n/ui-*.json` は **DB02 / P303 と共有領域**。今回 P302 が touch するが、競合が起きやすい:

- 作業前: `git pull origin main` で最新化
- 作業中: 他セッションが同じファイルを触っていないか `git status` で確認
- 完了後: DB01 へ報告、push は DB01 がまとめる

### 2. battle_simulator 本体は touch しない

C5 持ち物統合 / Init-B メガ進化 等の進行中タスクと別の作業なので、battle_simulator.html 本体は **触らない**。今回は **ホームページからのリンク追加だけ**。

### 3. カード配置順の判断

既存カード並び:
```
[Pokédex] [Party Checker] [Move List] [Type Chart] [Making]
```

候補配置:
- (A) Type Chart の右隣: `[... Type Chart] [Battle Simulator] [Making]`
- (B) Party Checker の右隣: `[... Party Checker] [Battle Simulator] [Move List ...]`
- (C) 最後: `[... Making] [Battle Simulator]`

→ **(A) を推奨**(対戦準備ツール群でまとめる、Making は別カテゴリ)

迷ったら DB01 (私)に判断仰ぐか、(A) で進めて OK。

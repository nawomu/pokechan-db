# 相打ち(同時ひんし)の勝敗規則 — 権威裏取りメモ(2026-07-11)

タスクA.4のための調査。**自己出力をゴールデンにせず、権威ソース(ポケモンWiki等)に当てて確定した規則**。
実装は `real_battle.html`(gameOver/WIN・LOSE判定)+ エンジンの反動処理順にまたがるため、
B②(AIプリセット・host page編集)の統合後に着手する。

## 確定した規則(第3世代以降のビデオゲーム)

出典: ポケモンWiki「ターン」「だいばくはつ」「みちづれ」「反動技」/ WebSearch要約

1. **反動技での相打ち(フレアドライブ・すてみタックル等)** → **攻撃側(反動で倒れた側)の勝ち**。
   - 処理順: 攻撃がヒット→被弾側がひんし(告知)→特性/持ち物効果→反動ダメージ→攻撃側がひんし(告知)。
   - つまり「被弾側告知 → 反動 → 反動側告知」の順で、最後に反動側が倒れても**攻撃側が勝つ**。

2. **自爆技での全滅(じばく・だいばくはつ・いのちがけ)** → **使った側の負け**。
   - 自爆技で自分と相手の最後の1体が同時に倒れたら、技を使った側がバトルに負ける(特例)。

3. **みちづれでの全滅** → **みちづれを使った側の負け**。
   - みちづれ使用者は既に倒れており、その効果で相手も道連れ→両者全滅時は使用者側の負け。

## 実装時の要点(未着手・B②統合後)
- エンジン: 反動でのひんし告知順が「被弾側→反動→反動側」になっているか確認(なっていれば表示はOK)。
- host(real_battle.html): ターン終了時に**両者の最後の1体が同時ひんし**になったケースを検出し、
  上記の規則で勝者を決める。原因(反動 / 自爆技 / みちづれ)を区別する必要がある
  → 直前に使われた技の effects に `反動` / kind(じばく系・みちづれ)が含まれるかで判定。
- ゴールデン(正解)は上記の権威規則。sim/エンジンの自己出力を正解にしない。
